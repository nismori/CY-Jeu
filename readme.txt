Executer java -jar Main.jar niveauBob.txt
Vous pouvez modifier niveauBob.txt pour joueur dans un nouveau niveau (encore rudimentaire, c'est un bonus)

Commandes :

Compiler : 
1: javac *.java
2: jar cfm Main.jar META-INF/MANIFEST.MF .

Executer :
3: java -jar Main.jar argument si vous êtes en dessous du Monde 3 niveau 3
Sinon : 
java -jar Main.jar argument pour le mode par défaut ou
java -jar Main.jar pour le mode niveau par niveau

ATTENTION : Pour l'argument, ne prennez pas un niveau qui existe déjà (Niveau1.txt, Niveau2.txt, Niveau3.txt, Niveau4.txt et Niveau5.txt)
ATTENTION : Si vous voulez modifier les fichiers texte, garder la même taille de ligne pour chaque ligne.
